Getting Started

Excelsior is a "meta-framework" built upon some of best technologies currently avaliable on the web. Using these different vendor libraries as a base, allows developers to create cutting edge websites and applications at a much faster pace.

Read the Excelsior Web Framework Documentation to get started.
